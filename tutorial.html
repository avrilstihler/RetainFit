RetainFit - Guia para Usuários

Guia completo para uso do sistema de previsão de evasão em academias: RetainFit Analytics

1. Introdução

Este sistema foi desenvolvido para ajudar academias a prever a probabilidade de cancelamento de clientes com base em dados históricos e comportamentais.

Com ele, você pode:

  • Identificar clientes em risco de cancelamento
  • Tomar ações preventivas com recomendações personalizadas
  • Analisar padrões de cancelamento por plano, frequência e sazonalidade

2. Como Usar a Aplicação

2.1. Abas Principais

A aplicação possui 4 abas principais:

  • 📊 Previsão – Faça simulações individuais para avaliar o risco de cancelamento de um cliente específico.
  • 📈 Análise – Visualize gráficos e insights sobre cancelamentos por:
    • Estação do ano
    • Tipo de plano
    • Frequência semanal
  • 📋 Dados – Visualize todos os clientes cadastrados e filtre por:
    • Todos os clientes
    • Ativos
    • Cancelados
    • Alto risco
  • 🤖 Modelo – Importe novos dados para treinar o modelo e veja métricas de desempenho (acurácia, precisão, recall)

2.2. Passo a Passo para Uso

1. Importando Dados

  1. Vá para a aba "Modelo"
  2. Clique em "Importar Dados"
  3. Selecione o arquivo Excel (.xlsx ou .xls)
  4. Aguarde o processamento

✅ Pronto! O sistema já está treinado com os dados e pronto para análises.

2. Fazendo Previsões Individuais

  1. Preencha os campos:
    • Gênero
    • Idade
    • Meses como cliente
    • Frequência semanal
    • Tipo de plano
    • Estação do ano
    • Dias desde o último pagamento
  2. Clique em "Calcular Risco"

📌 O sistema mostrará:

  • % de risco de cancelamento
  • Nível de risco (Baixo, Médio, Alto)
  • Fatores de risco
  • Sugestões de ação

3. Analisando Dados

Na aba "Análise", você encontra:

  • 📊 Gráficos comparativos entre clientes ativos e cancelados
  • 🔍 Principais Insights, como:
    • Qual estação tem mais cancelamentos
    • Quais planos têm maior retenção
    • Como a frequência de uso afeta a evasão

3. Como Preparar a Tabela de Dados

3.1. Formato da Planilha

A tabela deve ser em Excel (.xlsx ou .xls) e conter as seguintes colunas:

Nome da Coluna Tipo de Dado Exemplo Obrigatório?
Gênero Texto "Masculino", "Feminino" ✅ Sim
Idade Número 25, 30, 40 ✅ Sim
Meses Número 3, 6, 12 ✅ Sim
Frequência Número 2 (vezes/semana) ✅ Sim
Plano Texto "Mensal", "Trimestral", "Anual" ✅ Sim
Estação Texto "Verão", "Inverno" ✅ Sim
Último Pagamento Data "15/03/2024" ✅ Sim
Status Texto "Ativo", "Cancelado" ✅ Sim

3.2. Exemplo de Planilha Válida

Gênero Idade Meses Frequência Plano Estação Último Pagamento Status
Feminino 28 6 3 Mensal Verão 10/03/2024 Ativo
Masculino 35 12 1 Anual Inverno 15/01/2024 Cancelado

4. Dicas para Melhor Utilização

  • 🔹 Atualize os dados regularmente para manter o modelo preciso.
  • 🔹 Use os filtros na aba "Dados" para encontrar clientes de alto risco.
  • 🔹 Acompanhe os insights na aba "Análise" para identificar padrões.
  • 🔹 Personalize as ações com base nas sugestões do sistema.
